# SETUP
On a MacOS: <br>
1 - Install brew: 
```sh
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" < /dev/null 2> /dev/null"
```
2 - Install youtube-dl and ffmpeg:
```sh
brew install youtube-dl ffmpeg
```

3 - Qt needs to be installed
